Feature Pipeline

The five products above are portfolio-level bets. But each one is built from a stack of features — and we generated 81 candidate features by running AI ideation agents across all 20 opportunity cells. After deduplication and scoring, 54 unique feature concepts remain, ranked on a 9-criterion rubric.

How features were scored. Each idea was evaluated on 9 criteria across four lenses: Desirability (problem severity 3x, buyer clarity 2x, time to value 2x), Feasibility (build feasibility 2x, data access 2x), Viability (business model 1x, defensibility 1x), and Builder Fit (agentic suitability 2x, greenfield advantage 1x). Maximum score: 72. Ideas scoring above 55 are Tier 1 candidates; 40–55 need one assumption validated; below 40 are parked.

42 of 54 ideas map directly to the five products as features or modules. The remaining 12 are independent concepts — some strong enough to be standalone products, others better as consulting deliverables.
